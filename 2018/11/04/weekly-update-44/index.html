一周记事：week 44

Posted on 2018-11-04 | In Notes | | Views:

这一周又急匆匆地度过了。

老实说这是忙碌的一周。因为周二 Introduction to Database 做 lab demo，周一晚上风风火火的加了原本要求里没有的 trigger，希望临阵磨枪，不快也光。惴惴不安到周二来到，才发现其实 prof 根本没问什么困难的问题，随便让写两个 queries 就给打发了。

如果说周二还算轻松过关，那么周三简直不要太辛苦。凌晨难以入眠，早上被迫告假，下午高压面试，晚上作业答辩。这一折腾下来很是累得够呛。

说说这次的面试吧，真是难。四位面试官车轮战，从前端到后端，数据结构到算法，JavaScript 到 C#，马不停蹄地问。更有操作系统、面向对象概念乘胜追击。最终情景问答和性格分析问题重重暴击。三个半小时的面试下来，我都感觉魂不附体了…

面试之后再次感到自己的水平真是差得还很远，很多基础的知识应该要懂得才是。虽然我在前端的学习上已颇为深入，但对与软件工程师所必备的基础技能上还得再多下些功夫才行。

周四、周五在公司里尝试了 Vue 的 l18n 配置。简单地说，不过是将职业申请的表单配置成双语版本罢了。但通过 l18n，我只需要通过配置文件便可以增添新的语言，而不需要在应用逻辑层再做条件判断，if else 之类的了。不得不说，这种以数据为驱动的开发方式深得我心。希望下周有空可以写一篇博文，介绍一下如何配置 Vue 里的 l18n。
